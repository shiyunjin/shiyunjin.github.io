    • Windows Server 2019使用KMS激活

    微软推出的Windows Server 2019操作系统已经没有办法使用slmgr进行激活了

    需要在PowerShell下使用DISM来进行激活，首先清确定你已经在局域网内搭建KMS服务器并设置dns到_vlmcs._tcp.lan

    选择对应的版本执行命令

    DISM /online /Set-edition:ServerStandard /ProductKey:N69G4-B89J2-4G8F4-WWYCC-J464C /AcceptEula

DISM /online /Set-edition:ServerDatacenter /ProductKey:WMDGN-G9PQG-XVVXX-R3X43-63DFG /AcceptEula

DISM /online /Set-edition:ServerSolution /ProductKey:WVDHN-86M7X-466P6-VHXV7-YY726 /AcceptEula
